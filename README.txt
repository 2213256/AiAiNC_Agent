To run the application:

1. Host the website using any hosting services.
	Example using Python:
		(IMPORTANT: Install Python)
		1.1. Go to project directory
		1.2. Open terminal
		1.3. Use the command: python -m http.server
2. Open a browser
3. If in desktop, go to localhost:8000
4. If in mobile, go to <IP Address of computer>:8000
